SNS に時間を使いすぎていませんか？
SocialHub は Twitter Mastodon Slack Tumblr Misskey を同時に閲覧・投稿が可能な高機能なクライアントアプリです。

Features

多様な SNS に対応

SocialHub は複数の SNS を跨いだ閲覧や投稿を可能にし、あなたのコミュニケーションをより効率的にします。

エレガントなデザイン

アプリは毎日使うもの、だからこそデザインは綺麗でわかりやすくあるべきです。SocialHub ではデザインを妥協しません。

用途に合わせたアカウント管理

マルチアカウント対応はもちろん、公私などの区別でグループ分けが可能で、シーンに合わせて使い分けが可能です。

SNS 固有の機能に対応

複数 SNS に対応していても、個々の対応を中途半端にはしません。Twitter では DM やトレンド等も完備しています。

高カスタマイズ性

高度にカスタマイズ可能なクライアントアプリを目指して実装を進めています。また道半ばですが、順次実装を進めています。

継続的なアップデート

SocialHub (2020-)

main About Project Contact

SocialHub 2020/2 に公開した Twitter, Mastodon, Slack, Tumblr, Misskey に対応する iOS 向けマルチ SNS クライアントアプリです。 詳しくは 紹介ページ をご覧ください。また AppStore より販売中です。 本アプリにて使用している J2ObjC を使った Java ライブラリ を OSS として公開しています。

使用技術
  • クライアント: Java, Swift, J2ObjC
  • サーバーサイド: Kotlin, SpringBoot, Google App Engine, Google Compute Engine
